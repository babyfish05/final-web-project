Cara Menggunakan Aplikasi Web Sistem Informasi Manajemen Produk
1. Halaman Login
Pengguna mengakses aplikasi melalui halaman login.
Pada halaman ini, pengguna harus memasukkan email dan kata sandi yang telah terdaftar pada sistem.
Jika data yang dimasukkan valid, pengguna akan diarahkan ke halaman dashboard. Apabila tidak valid, sistem akan menampilkan pesan kesalahan dan pengguna diminta mengulangi proses login.

2. Halaman Dashboard
Setelah berhasil login, pengguna akan masuk ke halaman Dashboard.
Dashboard berfungsi sebagai pusat informasi sistem yang menampilkan ringkasan data, antara lain:
-Total Kategori
-Total Brand
-Total Produk
-Total Satuan
-Total Stok
Selain itu, dashboard juga menampilkan informasi tanggal dan hari serta tabel Activity Stok Produk yang menunjukkan kondisi dan aktivitas stok barang. Halaman ini membantu admin memantau kondisi data secara cepat dan menyeluruh.

3. Menu Kategori
Menu Kategori digunakan untuk mengelola jenis atau kelompok produk.
Langkah penggunaan:
-Pilih menu Kategori pada sidebar.
-Klik tombol Tambah Kategori untuk menambahkan data baru.
-Isi nama kategori, lalu simpan data.
-Pengguna dapat melakukan edit atau hapus kategori sesuai kebutuhan.
-Kategori yang dibuat akan digunakan saat menambahkan data produk.

4. Menu Brand
Menu Brand digunakan untuk mengelola merek produk.
Langkah penggunaan:
-Pilih menu Brand.
-Klik Tambah Brand untuk menambahkan merek baru.
-Isi nama brand dan simpan data.
-Data brand dapat diedit atau dihapus apabila diperlukan.
-Satu brand dapat digunakan oleh lebih dari satu produk.

5. Menu Satuan
Menu Satuan digunakan untuk menentukan satuan produk.
Langkah penggunaan:
-Pilih menu Satuan.
-Klik Tambah Satuan.
-Masukkan nama satuan (contoh: pcs, box, botol).
-Simpan data satuan.
-Satuan ini akan digunakan dalam pendataan produk dan stok.

6. Menu Produk
Menu Produk merupakan bagian utama dalam sistem.
Langkah penggunaan:
-Pilih menu Produk.
-Klik Tambah Produk.
-Isi data produk yang meliputi:
-Nama produk
-Kategori
-Brand
Satuan
-Simpan data produk.
-Setiap produk harus terhubung dengan satu kategori, satu brand, dan satu satuan agar data tersimpan secara konsisten.

7. Menu Stok
Menu Stok digunakan untuk mengelola jumlah ketersediaan produk.
Langkah penggunaan:
-Pilih menu Stok.
-Pilih produk yang akan diperbarui stoknya.
-Masukkan jumlah stok masuk atau stok keluar.
-Simpan data.
-Perubahan stok akan secara otomatis memengaruhi jumlah stok produk dan ditampilkan pada dashboard.

8. Logout
Setelah selesai menggunakan sistem, pengguna dapat keluar dari aplikasi dengan memilih menu Logout.
Fitur ini bertujuan untuk menjaga keamanan sistem dan mencegah penyalahgunaan akses.
